# Full Stack Application

### Front End
- React
- next.js
- Typscript
- urql/Typeorm/Graphql
- GraphQL Code Generator
- chakra-ui
### Back End
- Node
- Redis
- Express/Express Session
- graphQL/Apollo Express server
- postgres
